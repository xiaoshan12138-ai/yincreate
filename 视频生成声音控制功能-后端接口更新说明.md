# 视频生成声音控制功能 - 后端接口更新说明

## 更新日期
2026-05-26

## 功能概述
在视频生成场景中，新增**声音开关**功能，允许用户选择生成有声视频或无声视频。

- **默认值**: 无声（`false`）
- **UI 位置**: 输入框底部，字符计数旁边
- **显示条件**: 仅在切换到「视频生成」类型时显示

---

## 接口变更

### 请求接口
`POST /generate?sync=true`

### 变更内容
在视频生成（`output_type: "video"`）的 `parameters` 对象中，新增 `sound` 字段。

### 参数说明

| 参数名 | 类型 | 位置 | 可选值 | 默认值 | 必填 | 说明 |
|--------|------|------|--------|--------|------|------|
| `sound` | boolean | `parameters.sound` | `true`, `false` | `false` | 否 | 是否生成有声视频。`true`=有声，`false`=无声 |

---

## 请求示例

### 无声视频（默认）

```json
{
    "output_type": "video",
    "model": "Kling 3.0",
    "vendor": "vendor_a",
    "feature": "text_to_video",
    "parameters": {
        "resolution": "1080P",
        "duration": 5,
        "ratio": "16:9",
        "sound": false
    },
    "prompt": "一只猫在追蝴蝶",
    "input_files": []
}
```

### 有声视频

```json
{
    "output_type": "video",
    "model": "Kling 3.0",
    "vendor": "vendor_a",
    "feature": "text_to_video",
    "parameters": {
        "resolution": "1080P",
        "duration": 5,
        "ratio": "16:9",
        "sound": true
    },
    "prompt": "海边日落，海浪拍打礁石",
    "input_files": []
}
```

### 带参考图的有声视频

```json
{
    "output_type": "video",
    "model": "kling_3_0_omni",
    "vendor": "vendor_a",
    "feature": "global_reference",
    "parameters": {
        "resolution": "1080P",
        "duration": 8,
        "ratio": "16:9",
        "sound": true
    },
    "prompt": "让 <<<image_1>>> 动起来",
    "input_files": [
        {
            "type": "image",
            "url": "data:image/jpeg;base64,/9j/4AAQ...",
            "purpose": "reference",
            "object_id": "image_1"
        }
    ]
}
```

## 前端实现细节

### 状态变量
```javascript
const videoSoundEnabled = ref(false)  // 默认无声
```

### UI 组件位置
位于输入框底部（`.input-footer`），字符计数右侧：

```vue
<button
  v-if="selectedType === 'video'"
  :class="['sound-toggle-btn', { active: videoSoundEnabled }]"
  @click="videoSoundEnabled = !videoSoundEnabled"
  :title="videoSoundEnabled ? '点击关闭声音' : '点击开启声音'"
>
  <i :data-lucide="videoSoundEnabled ? 'volume-2' : 'volume-x'"></i>
  {{ videoSoundEnabled ? '有声' : '无声' }}
</button>
```

### 请求构建逻辑
在 `buildGenerateRequest()` 函数中，当 `outputType === 'video'` 时：

```javascript
request.parameters = {
  resolution: params.resolution,
  duration: 5,
  ratio: params.ratio,
  sound: videoSoundEnabled.value  // 新增字段
}
```

---

## 后端处理建议

1. **参数接收**: 从 `request.body.parameters.sound` 获取布尔值
2. **默认处理**: 若字段缺失或为 null，默认按 `false`（无声）处理
3. **向下兼容**: 该字段为可选字段，不影响现有逻辑
4. **模型适配**: 根据不同模型厂商（vendor）的声音支持能力进行处理：
   - `vendor_a`: 支持原生声音生成
   - `vendor_b`: 可能需要后处理添加音效/背景音乐

## 测试用例

| 场景 | sound 值 | 预期结果 |
|------|----------|----------|
| 未手动切换（默认） | `false` | 生成无声视频 |
| 用户点击开启声音 | `true` | 生成有声视频 |
| 切换到图片生成 | 不传输该字段 | 正常生成图片，忽略声音参数 |
| 切换到数字人 | 不传输该字段 | 正常生成数字人视频 |
